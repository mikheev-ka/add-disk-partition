# Работа с дисками и разделами в Linux

Краткий конспект по созданию разделов и монтированию дисков.

## Добавление нового диска

Допустим, у вас появился новый диск `/dev/sdb`. Вы хотите создать на нём один раздел и смонтировать его в `/mnt/data`.

### 1. Создаём раздел

sudo fdisk /dev/sdb
Внутри fdisk:

Нажмите n для нового раздела

Выберите тип (primary), номер (1), используйте всё дисковое пространство (по умолчанию)

Нажмите w для записи таблицы разделов

### 2. Форматируем раздел (например, в ext4)
sudo mkfs.ext4 /dev/sdb1

### 3. Создаём точку монтирования и монтируем
sudo mkdir -p /mnt/data
sudo mount /dev/sdb1 /mnt/data

### 4. Добавляем в /etc/fstab для автоматического монтирования
Сначала узнаём UUID раздела:

sudo blkid /dev/sdb1
Затем добавляем строку в /etc/fstab:

UUID=ваш-uuid /mnt/data ext4 defaults 0 2
Или через имя устройства:

/dev/sdb1 /mnt/data ext4 defaults 0 2
Важное замечание
Теперь у вас есть раздел фиксированного размера, равного размеру диска. Если позже появится ещё один диск, вы не сможете просто расширить /mnt/data — придётся монтировать его отдельно.

# Полезные команды
### Посмотреть список дисков и разделов
lsblk

### Посмотреть информацию о дисках
sudo fdisk -l

### Посмотреть UUID разделов
sudo blkid

### Проверить свободное место
df -h
Альтернативы для гибкого управления
Если в будущем потребуется расширять пространство без создания новых точек монтирования, рассмотрите использование LVM (Logical Volume Manager).



# Пример создания LVM:

### Создаём физический том
sudo pvcreate /dev/sdb

### Создаём группу томов
sudo vgcreate data_vg /dev/sdb

### Создаём логический том
sudo lvcreate -l 100%FREE -n data_lv data_vg

### Форматируем и монтируем
sudo mkfs.ext4 /dev/data_vg/data_lv

sudo mount /dev/data_vg/data_lv /mnt/data

При добавлении нового диска (например, /dev/sdc):

sudo pvcreate /dev/sdc

sudo vgextend data_vg /dev/sdc

sudo lvextend -l +100%FREE /dev/data_vg/data_lv

sudo resize2fs /dev/data_vg/data_lv

Теперь /mnt/data увеличится без необходимости создавать новую точку монтирования.
