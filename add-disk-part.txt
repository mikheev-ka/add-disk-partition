Пример добавления обычного раздела на диск
Допустим, у вас появился новый диск /dev/sdb. Вы хотите создать на нём один раздел и смонтировать его в /mnt/data.

Создаём раздел:

sudo fdisk /dev/sdb
Внутри fdisk:

Нажмите n для нового раздела.

Выберите тип (primary), номер (1), используйте всё дисковое пространство (по умолчанию).

Нажмите w для записи таблицы разделов.

Форматируем раздел (например, в ext4):
sudo mkfs.ext4 /dev/sdb1

Создаём точку монтирования и монтируем:
sudo mkdir -p /mnt/data
sudo mount /dev/sdb1 /mnt/data
Добавляем в /etc/fstab для автоматического монтирования:

/dev/sdb1(или blkid) /mnt/data ext4 defaults 0 2
Теперь у вас есть раздел фиксированного размера, равного размеру диска. Если позже появится ещё один диск, вы не сможете просто расширить /mnt/data — придётся монтировать его отдельно
